## 零长数组

零长度数组在内核中的使用
零长度数组在内核中，一般以变长结构体的形式使用。今天我们就分析一下 Linux 内核中的 USB 驱动。在网卡驱动中，大家可能都比较熟悉一个名字：套接字缓冲区，即 socket buffer，用来传输网络数据包。同样，在 USB 驱动中，也有一个类似的东西，叫 URB，其全名为 USB request block，即 USB 请求块，用来传输 USB 数据包。
